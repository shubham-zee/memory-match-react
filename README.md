# Memory Match (React + Vite)

A simple emoji memory matching game built with React and Vite.

## Scripts

- `npm install`
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview built app

## Notes

- Built with functional components and hooks
- Difficulty selector (6/8/10 pairs)
- Flip animations, timer, moves counter
